.. py:function:: mwcs()

   Maximum Weighted Clique Search

   This function is a wrapper for the MWCS algorithm. It takes a graph as an
   input and returns a list of nodes that form a maximum weighted clique.

   :param G: The input graph.
   :type G: nx.Graph

   :returns: A list of nodes that form a maximum weighted clique.
   :rtype: list
